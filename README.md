Personal technical blog by **Huang Weiliang**, IEEE Senior Member and Platform Product Owner at Robert Bosch LLC.

## About

This blog serves as a comprehensive resource for embedded systems engineers, focusing on:

- **Linux Kernel Development**: Kernel internals, filesystems, device drivers
- **QNX Neutrino RTOS**: System architecture, development practices, porting
- **Automotive Software**: Cockpit systems, ADAS, display subsystems
- **Virtualization**: Hypervisors, VirtIO, mixed-criticality systems
- **Storage Subsystems**: UFS, M-PHY, storage performance optimization
- **Graphics & Display**: GPU integration, rendering pipelines, automotive HMI
- **Board Support Packages**: Low-level bring-up, bootloaders, hardware initialization
